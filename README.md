# 拼豆图案生成器

一个将图片转换为拼豆图案的Web应用，自动生成带有色号标识的拼豆图案。

## 功能特点

- 🖼️ **图片上传**：支持拖拽或点击上传图片（JPG、PNG、GIF格式）
- 🏷️ **品牌选择**：支持5种品牌（MARD、COCO、漫漫、盼盼、咪小窝）
- 📋 **色号表选择**：
  - 基础色号表：A、B、C、D、E、F、G、H、M（可多选）
  - 高级色号表：P(珠光)、Q(温变)、R(透明果冻水晶)、T(透明)、Y(夜光)、ZG(光变)（可选）
- 🎨 **自动颜色映射**：根据选择的品牌和色号表，将图片颜色自动映射到最接近的拼豆颜色
- 🔢 **色号标识**：每个网格显示对应品牌的色号
- 📊 **颜色统计**：自动统计每种颜色需要的豆子数量
- 📥 **图案下载**：可以下载生成的拼豆图案
- ⚙️ **可调节参数**：
  - 网格大小（20-100像素）
  - 显示/隐藏色号
  - 显示/隐藏网格线

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件（需要本地服务器或使用支持fetch的浏览器）
2. 点击上传区域或拖拽图片到上传区域
3. **选择品牌**：选择你使用的拼豆品牌（MARD、COCO、漫漫、盼盼、咪小窝）
4. **选择基础色号表**：至少选择一个基础色号表（A-H、M），可多选
5. **选择高级色号表**（可选）：根据需要选择高级色号选项
6. 调整网格大小和其他参数
7. 点击"生成拼豆图案"按钮
8. 查看生成的图案和颜色统计（显示对应品牌的色号）
9. 点击"下载图案"保存结果

## 色号表说明

### 基础色号表
- **A表**：26种颜色
- **B表**：32种颜色
- **C表**：29种颜色
- **D表**：26种颜色
- **E表**：24种颜色
- **F表**：25种颜色
- **G表**：21种颜色
- **H表**：23种颜色
- **M表**：15种颜色

### 高级色号表
- **P (珠光)**：23种颜色
- **Q (温变)**：5种颜色
- **R (透明果冻水晶)**：9种颜色
- **T (透明)**：1种颜色
- **Y (夜光)**：9种颜色
- **ZG (光变)**：8种颜色

## 品牌说明

系统支持5种品牌的色号对照：
- **MARD**
- **COCO**
- **漫漫**
- **盼盼**
- **咪小窝**

每个品牌在不同色号表中都有对应的色号标识。

## 技术实现

- 纯前端实现，使用Fetch API加载JSON数据
- 使用 Canvas API 进行图片处理
- 颜色匹配算法：欧几里得距离计算
- 动态颜色映射：根据用户选择的品牌和色号表构建颜色库
- 响应式设计，支持移动端

## 文件说明

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 核心功能实现
- `color_data.json` - 颜色数据（从Excel转换而来）
- `convert_excel_to_json.py` - Excel转JSON的转换脚本（可选）

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）

**注意**：由于使用了Fetch API加载JSON文件，建议使用本地服务器运行，或使用支持CORS的浏览器。如果直接在浏览器中打开HTML文件，某些浏览器可能会阻止加载JSON文件。
